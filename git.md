# Instruções para Iniciar e Atualizar um Repositório Git.

## Para atualizar arquivos existentes:
- **git init**
- **git pull**
- **git add .**
- **git commit -m "nome do commit"**
- **git push -u origin master**

## Para novos arquivos:
- **git init**
- **git add .**
- **git commit -m "nome do commit"**
- **git remote add origin {link do repositorio sem as chaves}**
- **git push -u origin master**
